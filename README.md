💡 Toggle Bulb Project

A simple and interactive Toggle Bulb web application built using HTML, CSS, and JavaScript. This project demonstrates DOM Manipulation and Event Handling by allowing users to turn a bulb ON and OFF through a button click or bulb interaction.

🚀 Features
Toggle bulb ON/OFF
Dynamic UI updates using DOM Manipulation
Interactive and user-friendly interface
Built with pure HTML, CSS, and JavaScript
🛠️ Technologies Used
HTML5
CSS3
JavaScript (DOM Manipulation)
